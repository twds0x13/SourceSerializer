# Multilingual Collaboration

SourceSerializer documentation uses Simplified Chinese as the source language with English as the translation target.

## Directory Structure

```
docs/          ← Chinese source
docs/en/       ← English translation (1:1 mapping)
```

## Translation Principles

- Chinese source changes must be synced to `docs/en/` in the same commit
- Terminology remains consistent: see [Glossary](./reference/glossary)
- Code blocks are not translated
- API signatures are not translated

## Participation

Non-Chinese speakers can participate via:
- Filing Issues for translation improvements
- Submitting PRs for English documentation
- Requesting new language support
