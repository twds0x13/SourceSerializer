# 多语言协作

SourceSerializer 文档以简体中文为源语言，英文为对译。

## 目录结构

```
docs/          ← 中文源
docs/en/       ← 英文对译（1:1 对应）
```

## 翻译原则

- 中文源修改时，必须同步修改 `docs/en/` 对应文件，同一 commit 提交
- 术语保持统一：见[术语表](./reference/glossary)
- 代码块不翻译
- API 签名不翻译

## 参与路径

非中英母语者可通过以下方式参与：
- 提交 Issue 讨论翻译改进
- 提交 PR 修改英文文档
- 请求新语言支持
